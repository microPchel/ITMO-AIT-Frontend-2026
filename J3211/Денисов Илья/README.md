# Task Search

Task Search — это небольшое frontend-приложение для просмотра задач, поиска подходящих ролей и записи на свободные позиции внутри проектов.

Приложение сделано как многостраничный сайт на чистом JavaScript без backend-части. Основной упор сделан на работу с DOM, обработку пользовательских действий, фильтрацию данных, модальные окна и сохранение состояния через `localStorage`.

## О проекте

Сайт моделирует простую систему поиска задач внутри команды или компании. Пользователь может:

- зарегистрироваться или войти в систему;
- открыть список задач;
- искать задачи по названию, описанию или постановщику;
- фильтровать задачи по статусу и приоритету;
- сортировать карточки;
- открывать подробную информацию о задаче в модальном окне;
- записываться на свободные должности;
- просматривать свои отклики в личном кабинете.

Проект полностью клиентский и не использует сервер или базу данных. Все данные берутся из локальных JavaScript-файлов и `localStorage`.

## Основной функционал

### 1. Стартовая страница
На стартовой странице пользователь может выбрать один из двух сценариев:
- вход в систему;
- регистрация нового аккаунта.

### 2. Вход
Страница входа позволяет авторизоваться по email и паролю.  
Если поля пустые или данные введены неверно, показывается уведомление об ошибке.

### 3. Регистрация
Пользователь может создать новый аккаунт, указав:
- имя;
- фамилию;
- email;
- пароль;
- повтор пароля.

После успешной регистрации новый пользователь сохраняется в `localStorage`.

### 4. Страница задач
На основной странице выводятся карточки задач. Для каждой задачи отображаются:
- изображение;
- название;
- статус;
- приоритет;
- постановщик.

Также доступны:
- поиск;
- фильтрация по статусу;
- фильтрация по приоритету;
- сортировка по имени постановщика.

### 5. Подробная информация о задаче
При нажатии на карточку открывается модальное окно, где можно посмотреть:
- описание задачи;
- срок;
- участников;
- свободные роли;
- список файлов;
- комментарии.

Если у задачи есть свободные должности, пользователь может записаться на выбранную роль.

### 6. Личный кабинет
Все выбранные пользователем роли сохраняются и отображаются в личном кабинете.  
Для каждой сохранённой заявки показываются:
- задача;
- выбранная роль;
- статус;
- приоритет;
- постановщик.

## Хранение данных

| Источник       | Что хранится                              | Ключи в localStorage     |
|----------------|--------------------------------------------|---------------------------|
| JS-файлы       | задачи, роли, комментарии, файлы, базовые пользователи | —                         |
| localStorage   | зарегистрированные пользователи            | `users`                   |

## Как запустить

Проект статический — нужен только локальный сервер.

### Вариант 1 — Python

```bash
# из корня проекта
py -m http.server 5500
# или
python -m http.server 5500
# или
python3 -m http.server 5500
→ открыть в браузере:
http://localhost:5500/index.html
Вариант 2 — VS Code + Live Server
Просто правой кнопкой → «Open with Live Server»

Сценарий использования

Открыть index.html
Войти или зарегистрироваться
Перейти к списку задач
Найти задачу (поиск / фильтры / сортировка)
Открыть подробности
Записаться на свободную роль
Перейти в личный кабинет → увидеть свою заявку


Особенности реализации

Чистый JavaScript без фреймворков
Динамический рендер карточек и модального окна
Поиск, фильтрация и сортировка — без перезагрузки страницы
Сохранение авторизации и откликов между сессиями
Bootstrap 5 + минимальные собственные стили
Модульная структура JS (ES6 import/export)
